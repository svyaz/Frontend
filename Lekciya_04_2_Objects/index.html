Задача «Работа с объектами»

  • Создайте массив объектов-стран (пусть будет несколько стран):
    • У страны есть название и список городов (пусть будет по несколько городов)
    • У города есть название и численность населения
  • Найдите страну/страны с максимальным количеством городов
  • Получите объект с информацией по всем странам такого вида: ключ – название страны, значение – суммарная численность по стране
  • Оформите код в виде функций
